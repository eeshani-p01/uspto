REQUEST ACCESS

The Access Request platform allows you view what access is needed for resource in USPTO environment. Some accounts might be subject to an approval chain before you are granted and are able to connect to the applications or environments. Additional features include ability to view all the applications and environments user information to gain access as well as links associated with the application.

Access request allows you to request the following types of items:

  • Application access request
  • Access Profile
  • Contact information to make the request
  • Links to the application
